# slider

## Чистый JavaScript
______
Проект предоставляет возможность демонстрации результата выполнения услуг (дизайн интереьеров).
Управление сменой слайдов происходит через:
- стрелки - кнопки,
- точки - доты,
- ссылки - описания.
